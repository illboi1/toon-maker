# Toon Maker
Convert a video into cartoon style.



## 사용자 입력

#### [TAB키] 모드 변환
- 모드 1: 카툰 스타일 뷰 (Cartoon Style)
- 모드 2: 엣지 검출 뷰 (Edge Detection)

#### [＋/－키] 스케일 확대 / 축소 (10%p)
